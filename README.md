# تطبيق لوحة معلومات الطاقة الشمسية ☀️

هذا التطبيق يعرض معلومات عن الطاقة الشمسية.

## أولًا: لازم يكون عندك برنامج بايثون (Python) على جهازك

### لو عندك جهاز ويندوز (Windows)

1.  اضغط هنا عشان تنزل برنامج بايثون: [https://www.python.org/downloads/windows/](https://www.python.org/downloads/windows/) ودور على أحدث إصدار مكتوب جنبه Python 3 ونزله.
2.  بعد ما البرنامج ينزل، شغله عشان يتثبت. **مهم جدًا تعلم علامة صح على الخيار اللي بيقول "Add Python to PATH"** وأنت بتثبت البرنامج.
3.  بعد ما التثبيت يخلص، افتح برنامج اسمه "موجه الأوامر" (Command Prompt) اكتب في البحث بتاع قائمة ابدأ "cmd" وافتحه.
4.  في البرنامج اللي فتح ده، اكتب الأمر ده بالظبط ودوس إنتر: `python --version`. لو ظهر رقم، يبقى بايثون متثبت صح.

### لو عندك جهاز ماك (macOS)

1.  اضغط هنا عشان تنزل برنامج بايثون: [https://www.python.org/downloads/macos/](https://www.python.org/downloads/macos/) ودور على أحدث إصدار مكتوب جنبه Python 3 ونزله.
2.  بعد ما البرنامج ينزل، شغله عشان يتثبت، واتبع التعليمات اللي هتظهرلك.
3.  بعد ما التثبيت يخلص، افتح برنامج اسمه "Terminal" هتلاقيه في Applications وبعدين Utilities.
4.  في البرنامج اللي فتح ده، اكتب الأمر ده بالظبط ودوس إنتر: `python3 --version` أو ممكن تجرب تكتب `python --version`. لو ظهر رقم، يبقى بايثون متثبت صح.

## ثانيًا: هنثبت شوية حاجات عشان التطبيق يشتغل

1.  **احفظ الكود:** خد الكود اللي أنا اديتهولك ده واحفظه في ملف لوحده وسميه أي اسم بس لازم يكون آخره `.py`، زي مثلًا `solar_app.py`. حط الملف ده في أي مكان سهل تفتحه بعد كده، زي سطح المكتب مثلًا في فولدر جديد اسمه "SolarApp".
2.  **افتح برنامج موجه الأوامر (Windows) أو Terminal (macOS) تاني.**
3.  **روح للمكان اللي حفظت فيه الملف:** استخدم الأمر `cd` عشان تدخل جوة الفولدر اللي فيه الملف. لو حفظته على سطح المكتب في فولدر اسمه "SolarApp"، هتكتب كده وتدوس إنتر:
    * **ويندوز:** `cd Desktop\SolarApp`
    * **ماك:** `cd Desktop/SolarApp`
4.  **ثبت البرامج اللي التطبيق محتاجها:** اكتب الأوامر دي بالظبط واحد ورا التاني ودوس إنتر بعد كل واحد:

    ```bash
    pip install streamlit
    pip install pandas
    pip install numpy
    pip install matplotlib
    pip install pvlib
    pip install pytz
    pip install openpyxl
    pip install xlsxwriter
    ```

    استنى لما كل أمر يخلص تحميل وتثبيت قبل ما تكتب الأمر اللي بعده.

## ثالثًا: شغل التطبيق بقى

1.  تأكد إنك لسه فاتح برنامج موجه الأوامر أو Terminal وواقف جوة الفولدر اللي فيه ملف التطبيق (`solar_app.py`).
2.  اكتب الأمر ده بالظبط ودوس إنتر:

    ```bash
    streamlit run solar_app.py
    ```

    المفروض دلوقتي صفحة إنترنت جديدة تفتح لوحدها فيها التطبيق. لو مفتحتش، بص في برنامج موجه الأوامر أو Terminal هتلاقي رابط كده زي `http://localhost:8501`، خده واكتبه في المتصفح بتاعك.

## طريقة استخدام التطبيق

* هتلاقي قائمة على الشمال، دوس على اللي يعجبك: "Calculator" عشان تحسب، "Equations" عشان تشوف القوانين، "About Solar Energy" عشان تعرف معلومات عن الطاقة الشمسية.
* لو دوست على "Calculator"، هتلاقي خانات تدخل فيها أرقام، وبعدين تدوس على زر "Calculate Solar Radiation".
* هتشوف نتائج ورسومات وجدول فيه المعلومات.
* لو عايز تحفظ الجدول ده في ملف إكسل، دوس على زر "Download Excel Report".

## لو حصلت مشكلة

* **لو الأمر `streamlit` مش شغال:** يبقى ممكن يكون متثبتش صح، حاول تقفل برنامج موجه الأوامر أو Terminal وتفتحه تاني وتجرب الأمر تاني.
* **لو ظهر كلام أحمر كتير:** يبقى ممكن يكون فيه برنامج من اللي التطبيق محتاجها متثبتش صح، حاول تكتب الأمر بتاعه تاني اللي في خطوة تثبيت البرامج.
